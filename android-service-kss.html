Service

A service is a component which runs in the background without direct interaction with the user.

As the service has no user interface, it is not bound to the lifecycle of an activity.


Uses:

  • Playing music
  • Internet downloading
  • heavy processing of data

Implementing Service

To create a service, the class should implement the android.app.Service interface.


import android.app.Service;
import android.content.Intent;
import android.os.IBinder;
import android.support.annotation.Nullable;

public class MyService extends Service {
	...
}

Declaring the Service

In order for the Service to work, the implementing class should be declared in the AndroidManifest file.


<application ... >
	<service android:name=".MyService">
</application>
